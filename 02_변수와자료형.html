변수 (variable)

        *메모리에 값을 계속 바꿔가면서 저장할 수 있는 공간
        변하는 수
        * 변수에 이름을 붙이고 (변수명) 원하는 값을 넣어주면
            변수명이 필요한 곳에서 변수명을 호출해서 지정된 값을 사용할 수 있음


        
변수명 이름 작명 규칙

        - 의미 있는 이름 작성(a, b, c 이런식으로 작성하는 것 x)
        - 대, 소문자 작성
        - 특수 문자는 _(언더스코어) 만 사용 가능(단, 척 글자는 제외)
        - 중간에 대문자 작성해주는 카멜표기법 작성하는

        [카멜 표기법 (Camel Case)]
        문장에서 여러 단어를 변수명으로 작성할 때 이어지는 첫 번째 단어를 대문자로 작성
        member name => memberNeam

Java Script 변수의 종류와 특징

        var   : 변수, 변수명 중복o [덮어쓰기 가능]
            -> 변수명 재사용 가능, 변수 안에 있는 내용 변경 가능
        let   : 변수, 변수명 중복 x
            -> 변수명 재사용 불가능, 변수명 안에 있는 내용 변경 가능
        const : 상수, 변수명 중복 x
            -> 모두 불가
