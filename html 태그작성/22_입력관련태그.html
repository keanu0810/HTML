form 태그 / input 태그

form 태그

        - 입력 양식을 작성한은 영역(공간)
        - 내부에 작성된 입력 밧을 지정된 공간으로 제출하는 역할

        속성
        1. action             : 제출된 공간을 지정하는 속성(주소를 작성)
        2. method(방법 / 방식) : 설정된 공간을 ㅗ데이터를 전달하는 방식
        3. name               : 자바스크립에서 사용 form 폼우ㅏ 아룸울 자종허눈 속성
        4.target               제출된 페이지를 현재 또는 새 탭으로 열기 설정
